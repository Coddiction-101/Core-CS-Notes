## 📦 What's Inside 

| Module       | What You Do                                                                                                                                                                              |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Module 1** | 6 exercises on your real machine — ipconfig, ping cascade, traceroute analysis, DNS investigation (A/MX/NS records), ARP table, netstat active connections                               |
| **Module 2** | 50 subnetting problems — Section A (basic, 20 problems + full answer key), Section B (design subnets, 15 problems + answers), Section C (applied/VLSM/real-world, 15 problems + answers) |
| **Module 3** | Packet Tracer — star topology + DHCP server + simulation mode to watch packets travel                                                                                                    |
| **Module 4** | Packet Tracer — two LANs + router + the KEY observation: MAC changes every hop, IP stays same                                                                                            |
| **Module 5** | Packet Tracer — VLANs + trunk + Router-on-a-Stick — prove isolation then prove inter-VLAN routing                                                                                        |
| **Module 6** | Packet Tracer — static routing first, then replace with OSPF — observe the difference                                                                                                    |
| **Module 7** | Wireshark — 5 captures: DNS, TCP handshake, HTTP GET/200OK, ARP broadcast/reply, DHCP DORA                                                                                               |
| **Module 8** | Full college network design — paper design first, then Packet Tracer build, full test checklist, 5 design challenge questions                                                            |
